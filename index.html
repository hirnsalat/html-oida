HTML, oida. A slightly less minimalist Pandoc HTML template.

You know what’s great? Pandoc. You know what’s slightly less great? The readability of it’s default HTML template. So I copied over the 7 CSS declarations from the Better beeping Website. You’re welcome.

How do I use this?

Refer to the Pandoc documentation for now. I’ll update this section when I get around to it.

To use this as your default template, move html-oida.html to the templates folder of your Pandoc user data directory (search for --data-dir) and rename it to default.html5.
